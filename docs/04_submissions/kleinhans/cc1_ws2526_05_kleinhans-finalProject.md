## Learnings
I gained a lot more practice in JavaScrip and I am very happy to learn ThreeJS because it made me feel motivated again to do some web development! 
The biggest challenge of the Hühnerfestival Website was to build the navigation betwenn the three modes only by animating the camera. Getting this system working callenged also other implementations, for instance the artist cube, which took a lot of trail and errors to adjust in the right angle. Because I spent a lot of time debugging and solving technical issues, some of my original ideas didn’t make it into the final version. In the end I had way to little time to finally style other elements, what was a bit unsatisfying. Still I really enyojed the process and had a lot of fun during coding anyway.
